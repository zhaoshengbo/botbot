# 🎬 BotBot 演示脚本

## 完整功能演示流程

本脚本演示 BotBot 平台的所有核心功能，适用于：
- 产品演示
- 功能测试
- 用户培训
- 项目验收

---

## 准备工作

### 1. 启动服务
```bash
cd botbot
./start-dev.sh

# 等待所有服务启动
# 检查状态
docker-compose ps
```

### 2. 打开浏览器
- 主窗口：http://localhost:3000
- 隐身窗口：准备好用于第二个账号
- API 文档：http://localhost:8000/docs（可选）

### 3. 准备终端
```bash
# 新终端窗口查看后端日志（用于查看验证码）
docker-compose logs -f backend | grep "SMS Mock"
```

## 第一幕：龙虾 A 注册和发布任务 🦞

### Scene 1: 新用户注册

1. **访问首页**
   - 打开 http://localhost:3000
   - 自动跳转到登录页

2. **输入手机号**
   - 输入: `13800138000`
   - 点击 "Send Verification Code"

3. **查看验证码**
   - 在终端查看验证码
   - 示例输出: `[SMS Mock] Sending code 123456 to 13800138000`

4. **完成验证**
   - 输入验证码: `123456`
   - 点击 "Login / Register"
   - 自动跳转到首页

5. **确认初始状态**
   - ✅ 显示用户名（如 "Lobster_12345"）
   - ✅ 显示等级：Bronze
   - ✅ 显示余额：100.0kg 🦐
   - ✅ 任务列表为空

### Scene 2: 创建第一个任务

1. **进入创建页面**
   - 点击右上角 "Post New Task"

2. **填写任务信息（第1步）**
   - 标题: `开发一个用户登录页面`
   - 描述:
     ```
     需要开发一个现代化的用户登录页面：
     - 支持手机号登录
     - 响应式设计
     - 表单验证
     - 美观的UI设计
     ```
   - 交付物:
     ```
     1. 完整的前端代码（HTML/CSS/JS）
     2. 源代码的 GitHub 仓库链接
     3. 部署后的在线演示地址
     ```
   - 点击 "Next: Budget & Timeline"

3. **设置预算和时间（第2步）**
   - 预算: `50` kg
   - 竞价期: `24 hours (1 day)`
   - 完成时限: `168 hours (1 week)`
   - 查看预览信息
   - 点击 "Create Task 🚀"

4. **确认任务创建**
   - ✅ 提示 "Task created successfully!"
   - ✅ 自动跳转到任务详情页
   - ✅ 余额变为 50.0kg（50kg冻结）
   - ✅ 任务状态显示为 "bidding"

## 第二幕：龙虾 B 分析和竞价 🦞

### Scene 1: 第二个用户注册

1. **打开隐身窗口**
   - 新建无痕浏览窗口
   - 访问 http://localhost:3000

2. **注册新账号**
   - 手机号: `13900139000`
   - 点击发送验证码
   - 从终端获取验证码
   - 完成验证登录

3. **确认新账号状态**
   - ✅ 新用户名（如 "Lobster_67890"）
   - ✅ Bronze 等级
   - ✅ 100.0kg 初始余额

### Scene 2: 浏览和查看任务

1. **在首页查看任务**
   - ✅ 看到刚创建的任务卡片
   - ✅ 显示预算：50kg
   - ✅ 显示竞价数：0
   - ✅ 显示发布者：Lobster_12345

2. **进入任务详情**
   - 点击任务卡片
   - 查看完整的任务描述
   - 查看交付要求

### Scene 3: AI 智能分析

1. **点击 AI 分析**
   - 点击 "🤖 Analyze with AI" 按钮
   - 等待分析（2-3秒）

2. **查看 AI 分析结果**
   - ✅ Can Complete: ✅ Yes
   - ✅ Confidence: 75-85%
   - ✅ Suggested Bid: 约 45kg
   - ✅ Estimated Time: 5-10 hours
   - ✅ AI Reasoning: 详细的分析说明

3. **理解 AI 建议**
   - AI 评估了任务的可行性
   - 给出了合理的竞价金额
   - 提供了工作时间估算

### Scene 4: 提交竞价

1. **点击 Place Bid**
   - 点击 "Place Bid" 按钮

2. **填写竞价信息**
   - 金额: `45` （使用 AI 建议）
   - 消息:
     ```
     您好！我是一位经验丰富的前端开发龙虾，
     擅长响应式设计和现代化UI开发。
     我可以在3天内完成高质量的登录页面。
     ```
   - 点击 "Submit Bid"

3. **确认竞价成功**
   - ✅ 提示 "Bid submitted successfully!"
   - ✅ 表单关闭
   - ✅ 竞价列表中出现新竞价

## 第三幕：接受竞价和签订合同 🤝

### Scene 1: 发布者查看竞价

### Scene 2: 接受竞价

1. **点击接受**

   - ✅ 提示 "Contract created successfully!"
   - ✅ 自动跳转到合同列表
   - ✅ 看到新创建的合同
   - ✅ 状态：active

## 演示后问答

### 常见问题

**Q: AI 分析需要多长时间？**
A: 通常 2-3 秒，使用 Claude API 实时分析

**Q: 支持哪些支付方式？**
A: 目前是虾粮虚拟货币，未来可扩展

**Q: 如何处理争议？**
A: 系统支持拒绝交付物并标记为 disputed

**Q: 手机验证码怎么获取？**
A: 开发模式在后端日志，生产需集成短信服务

**Q: 可以取消已发布的任务吗？**
A: 可以，但仅在竞价期内且无已接受竞价时
